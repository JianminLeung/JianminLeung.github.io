CSS | 半透明框

Veröffentlicht am 2017-10-16

背景知识：

RGBA\HSLA颜色

颜色（CSS权威第四章）

1、web安全颜色：20%、51、33（十六进制）的倍数。

指在256色计算机系统上总能避免抖动的颜色
2、rgb()
rgba() 0:完全透明 1:完全不透明;
RGBA 颜色值得到以下浏览器的支持：IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+。
hsl()
HSL颜色值得到以下浏览器的支持：IE9+、 Firefox、 Chrome、 Safari 以及 Opera 10+。
HSL指的是 hue（色调）、 saturation（饱和度）、lightness（亮度）
Hue（0-360）：0（或360）是红色，120是绿色，240是蓝色
Saturation（0%-100%）：0%是灰色，100%是全彩
Lightness（0%-100%）：0%是黑色，100%是白色
hsla()
HSLA颜色值得到以下浏览器的支持：IE9+、 Firefox 3+、 Chrome、 Safari 以及 Opera 10+。

颜色和背景（CSS权威第九章）

前景色（文本）

1、color
值：|inherit
作用：改变颜色、影响边框（默认的前景色会应用到边框，可以用border-color覆盖color，同时利用边 框可以影响图像的前景色，图像不能被color影响）

背景：前景之下直到边框外边界的所有空间，边框是画在背景上的（一般不继承）

1、background-color
值：|transparent|inherit(IE8以下不支持继承属性）
2、background-image（图片尺寸又图片本身决定）
值：|none|inherit(IE8以下不支持继承属性）
元素图像默认地位于元素左上角，并在水平和垂直方向重复
注：设置一种可用的背景颜色，这样的话，假如背景图像不可用，页面也可获得良好的视觉效果。
3、background-repeat
值：repeat|repeat-x|repeat-y|no-repeat|inherit(IE8以下不支持继承属性）
注：repeat-x|repeat-y会把整个X|y轴平铺完整，即向两个方向进行平铺。
4、background-position（相对于元素内边距边界放置原图像，定义时先水平方向，然后垂直方向）
值：top|bottom|left|center|长度值|百分数值|inherit(IE8以下不支持继承属性）

①使用关键字，成对出现，若只定义了一个，第二个即为center。
②使用百分数值，成对出现，若只定义了一个，第二个为50%；
注意，百分数值同时应用于元素和图片，在出现百分数的时候要计算出偏移量，若出现偏移点不能一致时，使用差值。例如书P277，负分数值的不同效果。
③使用长度值，成对出现，若只定义了一个，第二个为50%，百分数和长度值可以混合使用；
长度值是从元素内边距区左上角开始偏移，即从一个左上角到另一个左上角的便宜。
5、background-attachment
值：scroll|fixed|inherit(IE8以下不支持继承属性）
注意：当文档向下滚动时，背景图像会随之滚动，若想固定图像位置，使用fixed，相对与可视区固定。
随着用户调整浏览器窗口的大小时，可视区会改变，导致背景的原图随着窗口大小改变位置。
还有，对于分页媒体，每个页面都生成自己的视窗，因此fixed关联背景在输出的每一页面上都应当显示。
汇总：background：只要至少出现一个值，其他省略值胃默认值，推荐使用

CSS3背景新特性：

1、background-image:可以同时加入多张图片，第一张在#最顶端，不同的照片可以设置不同的属性。
2、background-size
值：长度值|百分数|cover|contain
长度值为图片的宽度和高度，百分数为占父元素宽度和高度的百分比的大小，若只给出一个数，第二个为auto
cover:保持图像的纵横比缩放成将完全覆盖背景定位区域的最小大小
contain:保持图像的纵横比缩放成适合背景定位区域的最大大小
3、background-origin
值：padding-box|border-box|content-box
指定background-position 的相对位置，若background-attachment的属性为fixed，那么这个属性值无效
4、background-clip:指定背景描绘区域
值：padding-box|border-box|content-box
5、background-blend-mode(Ie不支持)
值：(https://www.w3cschool.cn/cssref/pr-background-blend-mode.html)

In the absence of a background on the html element, the body background will cover the page. If there is a background on the html element, the body background behaves just like any other element.

例子


border: 10px solid hsla(0, 0%, 100%, .5);
background: white;

记住：边框是在背景上描的边，白色背景上描半透明边框，与白色边框没有区别。

border: 10px solid hsla(0, 0%, 100%, .5);
background: white;
background-clip: padding-box;
